Documentation

The Interpreter Is Simple, But The Syntax Will Make You Go Crazy! So Read The Docs If You Are Confused:

How It Works

When You Run Your Program The Interpreter Initializes An Empty Array. You Write 1 Or More Character Commands To Read And Write From This Array.

Learning Commands

There Are 11 Commands In This Language. Heres How To Use Them:

  • / - Moves The Selector Position Up
  • \ - Moves The Selector Position Down
  • . - Print The Array
  • % - Print The Current Position Value
  • | - Print New Line
  • # - Clears The Array But Position Is Saved
  • ? - Print Current Position
  • $pos; - Set To Provided Position Value
  • =text; - Sets Array Value At Current Position
  • :code; - Sets Array Value But Evaluates JS
  • [text] - Code Comments
